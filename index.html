python学習前に知っておいた方が良い事 10選 - 1.拡張子

ポイント


  • 拡張子はPC/Appがファイルの種類を識別するためにファイルの名前の末尾につけられる文字列
  • 「末尾につけられる文字列」は「. (ドット) +英数字3〜4文字」
    • PCでは拡張子が大文字(.CSV)でも動作するが別物として区別される環境もあるので小文字で統一するのがベスト
  • ファイル名はなるべく日本語ではなく半角英数字にする
  • ファイル名は記号を使わない
    • 文字を区切りたい時は「_ (アンダーバー, アンダースコア)」を使おう
    • 「- (ハイフン, マイナス)」も極力避けた方が良い
  • プログラミングでは普段使わない拡張子を多く使うようになる
    • 知らない拡張子が出てきたらどんな種類の拡張子が調べてみよう

python学習前に知っておいた方が良い事 10選 - 5.プログラミング言語

プログラミング言語 分類


コンパイル型言語

  • プログラムを実行する前に、機械が読める言語(0と1のデータ群)に翻訳する言語
  • プログラム実行前に翻訳を済ませておくため、プログラムの実行速度が速い
  • 学習に苦労しやすい
  • Java, C言語, C++, C#, Go言語, など

インタプリタ型言語

  • プログラムを実行しながら、機械が読める言語に少しずつ翻訳する言語
  • すぐにプログラムを実行できる
  • 学習しやすい
  • 実行速度が遅い
  • PHP, JavaScript, Python, Ruby, など

python学習前に知っておいた方が良い事 10選 - 9.文字化け

csvをエクセルで開くと
たまにこうなる理由


  • csvファイルの文字コード : UTF-8
    (Unicodeの符号を8ビット単位で表す)
  • エクセルのデフォルトのエンコーディング方式 : SHIFT_JIS
  • 文字コードが一致しない為Mojibakeが起きる
  • エンコーディング(encoding) : データを一定の規則に従って目的の情報に変換すること
  • エクセルでもエンコーディング方式をUTF-8に指定して読み込みを行えば文字化けしない

python学習前に知っておいた方が良い事 10選 - 9.文字化け

ポイント


  • 読み込みたいデータが保存されている文字コードを把握する
  • 同じ文字コードでエンコーディングして取り扱う
  • システム・プログラムによって特定の文字コードで無いと文字化けしてしまうモノもある
  • 文字化けして困ったらググってみよう！
  • 世界中で最もよく使われている文字コード : Unicode(UTF-8)

データの保存等はなるべくUTF-8で行う方が良い！
